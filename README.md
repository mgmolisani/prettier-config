# `@mgmolisani-private/prettier-config`

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ npm i -D @mgmolisani-private/prettier-config
```

**Edit `.prettierrc.js`**:

```javascript
module.exports = {
    ...require(`@mgmolisani-private/prettier-config`),
    // ...
}
```
